# Offshore Molds Website

Modern rebuild of the Offshore Molds marketing site, focused on U.S.-managed offshore plastic injection mold tooling, fast lead times, full-cycle process visibility, quality documentation, and RFQ conversion.

## Stack

- Next.js App Router
- React
- Tailwind CSS
- Lucide icons
- Vercel-ready multi-page marketing site

## Commands

```bash
npm run dev
npm run lint
npm run build
```

## Content Sources

The rebuild uses public content and image assets from the legacy Offshore Molds site, then restructures the messaging into a tighter homepage with dedicated pages for capabilities, process, quality, and contact.
